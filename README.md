# キックログ

## サイト概要
### サイトテーマ
メーカー別にサッカースパイクのレビューを書き合えるレビューサイト
​
### テーマを選んだ理由
サッカーが好きなのでサッカーについてのテーマを考えていました。<br>
サッカーショップサイトのスパイクページのレビュー欄を見たと時に、「履き心地」を中心としたレビューが多かったですが、プレイヤー目線で知りたい情報は「そのスパイクを履いた時にどんなプレーができるか」だと感じました。<br>
そこで、そのスパイクを履いた時の「キックのしやすさ、トラップのしやすさ、走りやすさ、疲れにくさ等」<br>
具体的なプレーにどう影響がでるかを共有し合えるサイトがあると多くのサッカープレイヤーの支えになりと思いこちらのサイトを作りたいと考えました。<br>

### ターゲットユーザ
・これからサッカーを始める人<br>
・サッカースパイクの買い替えを検討している人

### 主な利用シーン
・サッカースパイクの買い替えを検討をしている時<br>
・自分に合うスパイクを探したい時
